# Local Community Event Portal

## Description

A simple JavaScript web application developed for the Cognizant Java FSE SkillSpring Program.

## Features

- Display community events
- Search events
- Filter events by category
- Register for events
- Dynamic DOM Manipulation
- Form Validation
- JavaScript ES6 Features
- jQuery Demonstration

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery

## Folder Structure

M1 JavaScript
│
├── index.html
├── style.css
├── script.js
├── jquery.html
├── jquery.js
└── README.md

## How to Run

1. Open the project in VS Code.
2. Open `index.html` using Live Server.
3. Open `jquery.html` separately to view the jQuery demo.
