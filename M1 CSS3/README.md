# Local Community Event Portal - CSS3

## Description

A CSS3-based web application developed for the Cognizant Java FSE SkillSpring Program.

## Features

- Inline, Internal and External CSS
- CSS Selectors
- Typography
- Background Styling
- Tables
- Forms
- Box Model
- Multi-column Layout
- Responsive Design
- Media Queries
- CSS Grid

## Technologies Used

- HTML5
- CSS3

## Run

Open `index.html` in any web browser.
