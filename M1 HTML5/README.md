# Local Community Event Portal - HTML5

## Description

A simple HTML5 web application developed for the Cognizant Java FSE SkillSpring Program.

## Features

- Semantic HTML5
- Navigation
- Event Gallery
- Registration Form
- HTML5 Form Validation
- Event Handling
- Video Support
- Local Storage
- Session Storage
- Geolocation API
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Run

Open `index.html` in any modern web browser.
